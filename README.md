# Todo Management

- A todo management app, complete with login/register, full CRUD.
- There is an admin role, that allows you to add update and delete todos
- Basic roles can only complete them
- All users saved into a DB
- JWT Authentication

## Technologies used:
  - Spring Boot 3
  - Spring Security 6
  - JWT
  - Spring Data
  - MySQL
  - React
  - Bootstrap
  - JPA
